HOW TO USE:
controls: 

  W: forward

  A: strafe left

  S: backwards

  D: strafe right
  
  LEFT_SHIFT: Boost(any direction)

  TAB: free mouse
  
  EXIT: exit application
  
PS:
 (if you feel the render distance is too small you can modify the max amount of steps in the fragment shader)
